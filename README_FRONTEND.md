# economIA - Frontend (Site separado)

## Como rodar
Opção 1 (mais simples): abrir o arquivo
- Abra `frontend/index.html` no navegador (duplo clique).
- Não precisa instalar nada.

Opção 2 (recomendado para evitar restrições do navegador):
- Use um servidor local (ex.: Live Server no VSCode).

## Estrutura
- `frontend/index.html` → interface (formulários)
- `frontend/app.js` → cálculo e mensagens (simula o comportamento do Python no navegador)

## Funcionalidades implementadas
- **Análise geral financeira**: calcula percentuais, identifica se despesas estão acima do recomendado e mostra dicas (quando ativado).
- **Meta financeira**: calcula em quantos meses você atinge a meta.
